课程介绍

6周 - 360小时编程练习，Web开发从0到全栈

  • 第一周

    编程基本功训练

    快速的自我学习和解决问题的能力（学习方法、查找资料、5W1H、思维导图） 正确的编程思维（Tasking + Pipeline） 正确的编写代码（Clean Code） 用代码解决实际问题（如何将需求翻译成代码） 代码跟踪、调试技巧 自动化代码测试（Unit Test最佳实践） 使用版本控制工具管理自己的代码（Git + Github） 对必备基础知识和概念的了解和掌握（理论学习 + 讲代码 + 复盘） 命令行的使用

  • 第二周

    编程基本功训练

    Node.js 基础 模块化编程 ES6 非阻塞编程（Callback） 文件结构 面向对象基础：类、对象、实例，封装、继承，单一职责（可选）

  • 第三周

    理论知识与复杂技术栈

    Web开发基础 HTML5 + DOM + JQuery BootStrap 3（CSS） HTTP RESTful Web API + Express 用 Express 实现 RESTful 的几个标准 API 基于被搭建好的技术栈（能理解和描述）

  • 第四周

    理论知识与复杂技术栈

    React + Redux 前后端分离架构 HTTP协议及其应用 基于被搭建好的技术栈（能理解和描述） 自动化构建

  • 最后2周
    项目
    实战
